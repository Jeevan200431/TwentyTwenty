# TwentyTwenty Portfolio Web Application

This is a simple full-stack web application developed as part of the hiring assessment for Twenty20 Systems.

The application provides user registration and login functionality and displays a portfolio page after successful authentication.

---

## Features

- User Registration with Email and Password
- User Login with authentication
- Portfolio page displayed after login
- Displays logged-in user email
- Logout functionality
- Data stored in MongoDB
- Responsive design for desktop and mobile
- Deployed on Vercel

---

## Tech Stack

Frontend:

- JavaScript

## How the Application Works

1. User opens the landing page.
2. User registers using email and password.
3. User logs in using registered credentials.
4. After successful login, the user is redirected to the portfolio page.
5. Portfolio page displays user email and portfolio content.
6. User can log out and return to the landing page.
